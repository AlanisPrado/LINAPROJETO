# LINA PROJETO

## Instalação
1. Clone o repositório: `git clone https://github.com/usuario/meu-projeto.git`
2. Entre na pasta: `cd meu-projeto` onde for que voce instalo os arquivos
3. Instale dependências: `npm install`
4. Rode o projeto: `ng serve`

## Funcionalidades
- Cadastro de usuário
- Login
- Criar cardápio
- Consultar nutricionistas
